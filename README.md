STUDENT MARKS PREDICTION USING JAVA

1. OBJECTIVE

The objective of this project is to predict a student's marks based on the number of study hours entered by the user using a Java application.

2. SOFTWARE REQUIREMENTS

• Java JDK

• IntelliJ IDEA

3. PROJECT DESCRIPTION

This project predicts student marks based on study hours. The user enters the number of study hours, and the application calculates the predicted marks using a simple formula. The result is then displayed to the user.

4. WORKING

Step 1: User enters the number of study hours.

Step 2: The application calculates predicted marks using a predefined formula.

Step 3: If the predicted marks exceed 100, the value is limited to 100.

Step 4: The predicted marks are displayed on the screen.

5. OUTPUT

Example Input:

Study Hours: 5

Example Output:

Predicted Marks = 75.0

6. ADVANTAGES

• Easy to use

• Simple implementation

• Quick prediction of marks

• Helps understand basic prediction concepts

• Developed using Java

7. CONCLUSION

The Student Marks Prediction project was successfully developed using Java. The application predicts marks based on study hours and displays the result efficiently.
